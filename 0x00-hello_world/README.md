# This is the root folder for the 0x00-hello_world project.

~~~
0-preprocessor prints a .c file after it's been preprocessed.
1-compiler compiles a .c file but doesn't link it.
2-assembler generates the assembly code of a .c file.
3-name compiles a .c file to an executable program named 'cisfun'.
4-puts.c is a c program that prints "Programming is like building a multilingual puzzle\n".
5-printf.c is a c program that prints "with proper grammar, but the outcome is a piece of art,\n"
6-size.c is a c program that prints the size of various types.
100-intel generates the assembly of a .c file with intel syntax.
101-quote.c is a c program that prints exactly "and that piece of art is useful" - Dora Korpar, 2015-10-19\n" to standard error without using standard printing functions.
